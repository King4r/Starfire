What is Starfire about? +
Starfire is an all-in-one neural command center for Shopify store owners. We replace 5+ app subscriptions by combining AI-powered lifestyle photography, TikTok trend scanning, and automated email/social marketing into a single high-performance interface.
How to find and connect Shopify Admin API key? +
1. Go to your Shopify Admin > Settings > Apps and Sales Channels.
2. Click 'Develop Apps' > 'Create an App'.
3. Under 'Configuration', select 'Admin API Integration' and enable scopes for Products, Orders, and Customers.
4. Install the app and copy the 'Admin API Access Token' (starts with shpat_).
How to connect to Resend? +
1. Create a free account at Resend.com.
2. Go to the 'API Keys' tab in your Resend dashboard.
3. Create a new key with 'Full Access' and copy it.
4. Paste it into the Integration Key field in Starfire Settings.
